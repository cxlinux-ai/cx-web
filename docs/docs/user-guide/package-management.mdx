# Package Management

CX Linux provides a unified interface for package management across all major Linux distributions. Forget memorizing different package managersâ€”just describe what you need.

## Universal Package Operations

### Installing Packages

```bash
# Single package

# With specific version
```

## Cross-Distribution Compatibility

CX Linux translates your request to the appropriate package manager:

| Your Command | Ubuntu/Debian | Fedora/RHEL | Arch |
|-------------|---------------|-------------|------|
| "install git" | `apt install git` | `dnf install git` | `pacman -S git` |
| "remove nginx" | `apt remove nginx` | `dnf remove nginx` | `pacman -R nginx` |
| "update all" | `apt update && apt upgrade` | `dnf upgrade` | `pacman -Syu` |

### Package Name Translation

CX Linux handles package naming differences:

```bash
cx "install the apache web server"

# Ubuntu: apache2
# Fedora: httpd
# Arch: apache
```
