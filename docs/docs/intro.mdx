# Welcome to CX Linux

**CX Linux** is an AI-powered command-line tool that transforms how you interact with Linux. Instead of memorizing commands, just describe what you want in plain English.

```bash
# Traditional way
sudo apt update && sudo apt install nginx && sudo systemctl enable nginx && sudo systemctl start nginx

# CX way
cx "install nginx and set it up to start on boot"
```

## Your First Command

```bash

# Install something

# Preview first (dry-run)

```

### System Configuration

### Service Management
