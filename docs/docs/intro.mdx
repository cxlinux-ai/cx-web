# Welcome to Cortex Linux

**Cortex Linux** is an AI-powered command-line tool that transforms how you interact with Linux. Instead of memorizing commands, just describe what you want in plain English.

```bash
# Traditional way
sudo apt update && sudo apt install nginx && sudo systemctl enable nginx && sudo systemctl start nginx

# Cortex way
cortex "install nginx and set it up to start on boot"
```

## Your First Command

```bash
# Try it out
cortex "show system information"

# Install something
cortex "install htop"

# Preview first (dry-run)
cortex --dry-run "update all packages"
```

## What Can You Do?

### Package Management

```bash
cortex "install docker and docker-compose"
cortex "update nodejs to version 20"
cortex "remove unused packages"
```

### System Configuration

```bash
cortex "create user deploy with ssh access"
cortex "enable the firewall and allow port 443"
cortex "set up automatic security updates"
```

### Service Management

```bash
cortex "restart nginx and check its status"
cortex "disable bluetooth service"
cortex "show what services failed to start"
```

### Development Setup

```bash
cortex "set up python 3.11 with pip and virtualenv"
cortex "install rust development tools"
cortex "configure git with my name and email"
```
