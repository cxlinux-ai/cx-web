# CX Linux Contributor Referral Program

## Program Overview

**Launch Date:** January 2025
**Status:** LIVE
**Eligibility:** Discord Contributors Only (exclusive)

## Commission Structure

| Tier | Monthly Price | Referrer Cut (10%) |
|------|---------------|-------------------|
| Core+ | $20/system/mo | $2.00/system/mo |
| Pro | $99/mo | $9.90/mo |
| Enterprise | $299/mo | $29.90/mo |

**Commission Type:** Perpetual (lifetime of referred customer)

## How It Works

1. Contributors receive unique referral code via Discord #contributors
2. Share link: `https://cxlinux.ai/ref/{CODE}`
3. When referred user subscribes, referrer earns 10% monthly
4. Payouts processed monthly via PayPal/Wise (min $50 threshold)

## Referral Code Format

```
CX-{CONTRIBUTOR_HANDLE}-{4DIGIT}
```

Example: `CX-DEVJOHN-7842`

## Tracking

All referrals tracked in `referrals.json` with:
- Referrer code
- Referred user (anonymized ID)
- Subscription tier
- Sign-up date
- Monthly revenue
- Total earned

## Terms

1. 10% perpetual commission on all referred subscriptions
2. Commission continues as long as referred customer remains subscribed
3. No limit on number of referrals
4. Exclusive to verified Discord contributors
5. Fraud/abuse results in immediate termination
6. Minimum payout threshold: $50
7. Payouts on 15th of each month for previous month
