# Package Management

Cortex provides a unified interface for package management across all major Linux distributions. Forget memorizing different package managersâ€”just describe what you need.

## Universal Package Operations

### Installing Packages

```bash
# Single package
cortex "install nginx"

# Multiple packages
cortex "install git, curl, and wget"

# With specific version
cortex "install nodejs version 18"

# From specific repository
cortex "install docker from the official docker repo"
```

### Removing Packages

```bash
# Basic removal
cortex "remove nginx"

# Remove with configuration
cortex "completely remove nginx including config files"

# Remove unused dependencies
cortex "remove nginx and clean up unused dependencies"
```

### Updating Packages

```bash
# Update everything
cortex "update all packages"

# Update specific package
cortex "update nodejs to the latest version"

# Security updates only
cortex "apply security updates only"

# Update package lists
cortex "refresh package lists"
```

## Cross-Distribution Compatibility

Cortex translates your request to the appropriate package manager:

| Your Command | Ubuntu/Debian | Fedora/RHEL | Arch |
|-------------|---------------|-------------|------|
| "install git" | `apt install git` | `dnf install git` | `pacman -S git` |
| "remove nginx" | `apt remove nginx` | `dnf remove nginx` | `pacman -R nginx` |
| "update all" | `apt update && apt upgrade` | `dnf upgrade` | `pacman -Syu` |

### Package Name Translation

Cortex handles package naming differences:

```bash
cortex "install the apache web server"

# Ubuntu: apache2
# Fedora: httpd
# Arch: apache
```

## Troubleshooting

### Common Issues

#### Package Not Found

```bash
cortex "install superapp"
# Error: Package 'superapp' not found

# Try:
cortex "search for packages named like superapp"
cortex "what package provides superapp?"
```

#### Dependency Conflicts

```bash
# Let Cortex resolve
cortex "install package-a, resolve any conflicts automatically"

# Or investigate
cortex "why can't package-a and package-b be installed together?"
```

#### Locked Package Database

```bash
cortex "fix the locked package database"
cortex "another package manager is running, wait for it"
```

### Diagnostic Commands

```bash
# System check
cortex "check the package system health"

# Repair
cortex "repair the package database"

# Logs
cortex "show recent package manager errors"
```
