# Setting Up Dev Environments

This tutorial shows how to use Cortex to set up complete development environments. From basic language runtimes to full-stack frameworks, Cortex handles the complexity.

## Time Required

- Basic setup: 5 minutes
- Full environment: 15-30 minutes

## Prerequisites

- Cortex installed and configured
- Sudo access for package installation

## Python Development

### Basic Python Setup

```bash
cortex "set up python development environment"
```

Cortex will install:
- Python 3.x (latest stable)
- pip
- venv
- Common development tools

### Data Science Setup

```bash
cortex "set up python for data science with jupyter"
```

Includes:
- Python 3.x
- Jupyter Lab/Notebook
- NumPy, Pandas, Matplotlib
- scikit-learn

### Web Development (Django/Flask)

```bash
# Django
cortex "set up django development environment"

# Flask
cortex "set up flask development environment with postgres"
```

### Verify Setup

```bash
cortex "verify my python environment is working"
```

## Project-Specific Setup

### From README

```bash

```

Cortex analyzes:
- `package.json` for Node.js
- `requirements.txt` for Python
- `Cargo.toml` for Rust
- `go.mod` for Go
- etc.
